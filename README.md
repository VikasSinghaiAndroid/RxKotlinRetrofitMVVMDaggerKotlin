# RxKotlinRetrofitMVVMDaggerKotlin

This application shows how to fetch the huge data from server, parse the data and fill into RecyclerView with lazy loading.
Components used :
1. Kotlin language 
2. Retrofit
3. RxKotlin
4. RxAndroid
5. MVVM architecture
6. Dagger
7. RecyclerView
8. Pull to refresh
9. Picasso to load the image
